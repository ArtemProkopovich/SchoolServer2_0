Экшены загрузки файлов:
1. downloadJournal (Страница классного журнала, таблица учеников класса и их оценок по конкретному предмету)
Принимает:
[
	{
		"docType":"pdf"|"xls"|"csv", //Здесь и далее один из форматов на выбор пользователя.
		"subjectID":"1",
	},
...
]
Возвращает: InputStream с файлом.

2. downloadPupilSchedule (Расписание ученика на неделю)
Принимает:
[
	{
		"docType":"pdf"|"xls"|"csv",
		"pupilID":"1",
	},
]
Возвращает: InputStream с файлом.

3. downloadTeacherSchedule (Расписание учителя на неделю)
Принимает:
[
	{
		"docType":"pdf"|"xls"|"csv",
		"teacherID":"1",
	},
]
Возвращает: InputStream с файлом.

4. downloadPupilStats (Статистика ученика, таблица всех предметов ученика 
			и его оценок по ним и среднего балла по каждому предмету)
Принимает:
[
	{
		"docType":"pdf"|"xls"|"csv",
		"pupilID":"1",
	},
]
Возвращает: InputStream с файлом.

5. downloadPupilStats (Статистика рейтинга учеников класса, таблица со списком предметов класса 
			и средних оценок учеников по ним и рейтинг, как средняя оценка по всем предметам.)
Принимает:
[
	{
		"docType":"pdf"|"xls"|"csv",
		"classID":"1",
	},
...
]
Возвращает: InputStream с файлом.
